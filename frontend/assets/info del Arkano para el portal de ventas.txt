# Plan General Arkano - IA

## 1. Tipo de aplicación recomendada

Una **plataforma web administrativa + sitio público + app responsiva en Flutter**.

La idea sería tener:

**Sitio público**

* Página principal
* Talleres disponibles
* Casos de éxito
* Servicios
* Agenda de citas
* Contacto
* Blog / recursos

**Panel administrativo**

* Control de talleres
* Control de asistentes
* Citas
* Contactos / CRM
* Casos de éxito
* Galería de imágenes
* Pagos
* Reportes

**Frontend**

* Flutter Web primero
* Preparado para compilar después a Android/iOS

**Backend**

* Django + Django REST Framework
* PostgreSQL
* API para Flutter
* Panel admin interno

# 2. Módulos principales

## A. Módulo de talleres

Cada taller debe tener:

* Título
* Descripción corta
* Descripción completa
* Categoría
* Nivel: básico, intermedio, avanzado
* Modalidad: presencial, online, híbrido
* Fecha
* Hora
* Duración
* Precio
* Cupo máximo
* Lugar
* Instructor
* Estado: borrador, publicado, cerrado, finalizado
* Hasta **5 imágenes**
* Imagen principal
* Video promocional opcional
* Beneficios
* Temario
* Requisitos
* Público objetivo

---

## B. Galería por taller

Regla:

* Mínimo 1 imagen
* Máximo 5 imágenes
* Una imagen marcada como principal
* Orden de visualización
* Texto alternativo para SEO
* Compresión automática
* Vista tipo carrusel en Flutter

---

# 3. Arquitectura técnica

## Backend

Recomendado:

* Python
* Django
* Django REST Framework
* PostgreSQL
* JWT Authentication
* Django Admin
* Docker
* API REST

## Frontend

* Flutter Web
* Flutter Mobile preparado
* Riverpod o Bloc para estado
* Dio para consumir API
* GoRouter para navegación
* Material 3

## Storage de imágenes

Opciones:

### Opción inicial sencilla

Guardar imágenes en el servidor.

### Opción profesional

Usar almacenamiento tipo S3:

* AWS S3
* Cloudflare R2
* DigitalOcean Spaces
* Wasabi

Yo recomiendo **Cloudflare R2** por costo y facilidad.

# Recomendación final

Arkano - IA debe construirse como una **plataforma comercial inteligente**, no solo como página web.

La primera versión debe enfocarse en:

**vender talleres + captar contactos + mostrar autoridad + controlar asistentes.**

Después se convierte en:

**academia + CRM + consultoría + comunidad + plataforma IA.**
